# 🔄 页面刷新指南

## 问题现象

浏览器显示的是旧版本的界面（真实姓名 + 身份证号），而不是新版本（办税人 + 手机号 + 密码）。

## 原因

这是Next.js开发模式下的缓存问题。需要清除缓存并重新加载。

## 解决方案

### 方案1: 硬刷新浏览器（推荐）⭐

1. 在浏览器中按：
   - **Mac**: `Cmd + Shift + R`
   - **Windows/Linux**: `Ctrl + Shift + R`
   - **或**: `Ctrl + F5`

2. 如果还是不行，打开开发者工具（F12），右键点击刷新按钮，选择"清空缓存并硬刷新"

### 方案2: 清除Next.js缓存

```bash
# 停止开发服务器（Ctrl+C）
rm -rf .next
npm run dev
```

### 方案3: 清除浏览器缓存

1. Chrome浏览器：
   - 打开 `chrome://settings/clearBrowserData`
   - 选择"缓存的图片和文件"
   - 点击"清除数据"

2. 或者使用隐私模式/无痕模式：
   - Mac: `Cmd + Shift + N`
   - Windows: `Ctrl + Shift + N`

## 📞 获取帮助

如果按照上述方案仍然看到旧界面：

1. 检查文件是否最新：
```bash
cat app/user/bindcompany/page.tsx | grep "办税人手机号"
```

应该输出包含"办税人手机号"的行。

2. 查看浏览器控制台（F12）是否有错误

3. 确认访问的URL是：`http://localhost:3008/user/bindcompany`
